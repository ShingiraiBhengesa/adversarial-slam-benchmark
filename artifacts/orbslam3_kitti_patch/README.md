# ORB-SLAM3 KITTI Patch Artifacts

This directory contains curated small artifacts from the ORB-SLAM3 KITTI patch stress-test experiments.

Large raw outputs are intentionally not committed. Full trajectories, raw run folders, logs, generated image sequences, and datasets remain local under ignored paths such as:

- `results/`
- `logs/`
- `data/`
- `local_mechanism/`
