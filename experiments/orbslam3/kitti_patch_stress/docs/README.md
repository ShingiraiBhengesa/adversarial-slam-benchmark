# ORB-SLAM3 KITTI Patch Stress Test Docs

This folder contains the main writeups for the ORB-SLAM3 KITTI patch experiment.

## Main files

- `main_results.md`
  - main result summary and interpretation

- `mechanism.md`
  - ORB keypoint and match diagnostics explaining why checkerboard patches are different from black occlusion patches

- `official_kitti_devkit_results.md`
  - official KITTI odometry devkit cross-check results

- `severity_onset.md`
  - notes on when checkerboard patch failures begin and how severity changes results

- `trajectory_visualization.md`
  - notes on trajectory plots and visual comparison

## What this folder is for

Use this folder for clear experiment writeups.

Older intermediate notes are kept in:

- `../archive/`

Small result files are kept in:

- `../results_summary/`

Figures are kept in:

- `../figures/`
